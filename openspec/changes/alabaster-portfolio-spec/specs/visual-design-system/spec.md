## ADDED Requirements

### Requirement: Alabaster Color Palette
The portfolio SHALL use #faf9f5 as the base color, providing a solid, high-quality "paper" feel.

#### Scenario: Visual Consistency Check
- **WHEN** the user views any page
- **THEN** the background color MUST be #faf9f5 with a high-contrast, technical primary color for text.

### Requirement: Industrial Spec-Sheet Layout
The layout SHALL be inspired by technical spec sheets, featuring massive, bold headers and generous white space.

#### Scenario: Scanning for Hierarchy
- **WHEN** a user scrolls a project detail page
- **THEN** they MUST see hierarchical sections clearly separated by significant white space and industrial typography.

### Requirement: Technical Diagrams Over Imagery
The portfolio SHALL prioritize technical architectural diagrams (SVG/Canvas) over screenshots for system-level projects.

#### Scenario: Explaining Logic in Projects
- **WHEN** a user views a case study for Quix or Assistant Widget
- **THEN** they MUST see a technical diagram explaining the system architecture instead of a generic UI screenshot.

### Requirement: Refined Mechanical Motion
Any motion or animation in the portfolio SHALL use GSAP and feel intentional, mechanical, and refined.

#### Scenario: Interactive Feedback
- **WHEN** a user interacts with an element or scrolls
- **THEN** the resulting animation MUST be precise and subtle, avoiding playful or bouncy effects.
